# Web Projects 🌐

HTML and CSS projects from my BCA Web Technologies course.

## Files

### HTML Pages
-  `basic.html` — Basic HTML tags, headings, paragraphs, text formatting
-  `lists.html` — Ordered, unordered and nested lists with hyperlinks
-  `table.html` — Student marks table with styled rows
-  `form.html` — Student registration form with fieldset and iframe

### CSS
-  `style.css` — External stylesheet for all pages

## Concepts covered
- HTML tags and structure
- Forms — input, radio, checkbox, dropdown, textarea
- Tables — thead, tbody, tr, th, td
- Fieldset and legend
- Iframe and navbar
- External CSS styling
- Hover effects
- Box shadow and border radius

## How to run
Just open any .html file in your browser!
